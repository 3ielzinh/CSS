Redefinindo as propriedades padrões dos navegadores

Reset CSS é uma folha de estilos que irá sobrepor todas as propriedades de estilo CSS que os navegadores colocam por padrão nos elementros, ajudando com que os sites tenham o mesmo design em todos os navegadores.
